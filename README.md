# Análisis 1

En este análisis exploratorio se trabajó con un conjunto de datos sobre vegetales, utilizando exclusivamente RStudio como herramienta. El proceso incluyó la carga, limpieza y transformación de variables, así como la generación de estadísticas descriptivas y visualizaciones como histogramas y boxplots.
Se aplicaron pruebas de normalidad (Anderson-Darling) que mostraron que las variables Price (per kg) y Shelf Life (days) no siguen una distribución normal. También se evaluó la relación entre vida útil y precio a través de modelos de regresión lineal, observando un bajo coeficiente de determinación (R²), tanto con como sin valores atípicos, lo cual indica una débil relación lineal entre estas variables.
Este ejercicio permitió comprender cómo la diversidad del dataset influye en la variabilidad de los datos y resalta la importancia de considerar los outliers en los análisis. Todo el trabajo fue documentado con visualizaciones y conclusiones claras, y puede servir como base para un análisis más profundo en futuras etapas del proyecto.

Source data: https://www.kaggle.com/datasets/rudraprasadbhuyan/vegetables-dataset?resource=download
